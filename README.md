# NeonThread

Foro de discusión temático sobre cultura cyberpunk. Los usuarios pueden registrarse, crear hilos, comentar y explorar secciones dedicadas a videojuegos, libros, juegos de mesa y lore del género.

---

## Instrucciones de ejecución

### Requisitos
- [XAMPP](https://www.apachefriends.org/) con Apache activo
- PHP 8.0 o superior
- Navegador web moderno

### Pasos

1. Clona el repositorio dentro de la carpeta `htdocs` de XAMPP:
   ```bash
   git clone https://github.com/DamianRobles/NeonThread.git
   ```

2. Abre XAMPP y activa el módulo **Apache**.

3. La base de datos se genera automáticamente al acceder por primera vez al sitio. No se requiere configuración adicional.

4. Abre el navegador y accede a:
   ```
   http://localhost/NeonThread
   ```

## Tecnologías utilizadas

| Tecnología | Uso |
|---|---|
| PHP 8 | Lógica del servidor, sesiones y conexión a BD |

| CSS3 | Estilos personalizados con temática cyberpunk |
| Bootstrap 5 | Sistema de grid, componentes y diseño responsivo |
